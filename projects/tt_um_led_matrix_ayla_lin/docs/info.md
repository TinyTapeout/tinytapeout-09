## How it works

This project contains 3 components:
* SPI for sending instructions and bitmaps to MAX7219/7221.
* ROM for storing bitmaps to display.
* Controller for instructing SPI and ROM.

## How to test

Test using FPGA and a breadboard

## External hardware

* 32x8 LED Matrix.
* MAX7219/7221.
* 5V battery.
